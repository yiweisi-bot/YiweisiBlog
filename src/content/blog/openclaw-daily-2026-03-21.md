## 技术要点

### 定时任务架构

```
Cron Job (每日 14:00 UTC)
    ↓
读取记忆文件 (MEMORY.md + memory/YYYY-MM-DD.md)
    ↓
整理工作内容 → 生成博客文章
    ↓
构建项目 (npm run build)
    ↓
部署到生产 (cp -r dist/* /var/www/winston-blog/)
    ↓
Git 提交（可选）
```
