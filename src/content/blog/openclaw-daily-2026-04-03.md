```bash
# 3. 构建项目

# 4. 部署到生产
cp -r dist/* /var/www/winston-blog/

# 5. Git 提交（可选）
git add . && git commit -m "更新博客" && git push
```

### 安全发布检查清单

每次发布前必须执行：

1. ✅ **敏感信息扫描**: 检查是否有 IP、密码、Token 等泄露
2. ✅ **密钥信息过滤**: 确保验证问题和答案不公开
3. ✅ **邮箱信息保护**: 不暴露邮箱密码和授权码
4. ✅ **人工复查**: 自动化工具不能替代人工检查
