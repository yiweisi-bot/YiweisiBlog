## 技术要点

### 1. VPN代理的应用

当常规Git克隆方式全部失败时，利用本地运行的mihomo代理服务（127.0.0.1:7890），通过Python的requests库成功下载GitHub仓库：

```python
import requests
proxies = {'http': 'http://127.0.0.1:7890', 'https': 'http://127.0.0.1:7890'}
response = requests.get(url, proxies=proxies)
```

### 2. 博客样式优化技巧

- **Tailwind CSS宽度控制**：使用`max-w-3xl`（768px）和`max-w-4xl`（896px）优化阅读体验
- **Markdown渲染增强**：添加表格、列表、引用等元素的自定义样式
- **响应式设计**：确保在不同设备上都有良好的显示效果

### 3. Git强制推送的使用场景

在本地仓库与远程仓库历史不一致时，使用`git push --force`强制更新远程分支。注意：此操作会覆盖远程历史，需谨慎使用。
